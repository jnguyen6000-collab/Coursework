1. What did you learn about the security vulnerabilities of IoT devices through this exercise?


I learned how vulnerable an IoT device’s security is due to how easy it was to connect to it since it was close.
The thermostat that was connected showed me great details on temperature, schedule settings, HVAC settings, and alarm status. However, there was no password needed.
 All it took were a few clicks and I was easily able to connect to the thermostat.
The IoT device lacks any protection, making it extremely vulnerable to cyber threats.


2. How would you propose to protect the IoT device you selected from potential cyber-attacks?

My proposal to protect IoT devices is to add a strong password. Password is a simple way to protect your devices.
They can also add authentication to ensure that only authorized users can access the IoT device.
They can make sure the device’s software is updated and improve the hardware security to patch up any vulnerabilities within the IoT device.
