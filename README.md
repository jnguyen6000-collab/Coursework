# Cybersecurity Coursework

## Overview
This repository contains lab write-ups and reports completed during my 
Cybersecurity Administration degree at Holy Family University. Each file 
documents the tools used, findings, and key takeaways from hands-on work 
in areas including network security, ethical hacking, OSINT, and risk 
management.

## Skills Demonstrated

| Area | Labs |
|------|------|

| Vulnerability Assessment | Active Reconnaissance (Nessus/OpenVAS), Exploiting Vulnerable Hosts |
| Web Application Security | Web Application & Database Attacks (SQLi, XSS) |
| Wireless Security | Exploiting Wireless Vulnerabilities |
| OSINT | Shodan, Wigle, Wunderfind, HaveIBeenPwned |
| System Administration | Linux Write-Up, Managing Group Policy |
